1) XAMPP muss installiert werden und der gepackte Ordner in die htdocs Datei entpackt werden. 

2) XAMPP muss gestartet werden, 
"Apache" und "MySQL" müssen gestartet werden. 


Datenbank: 
Im XAMPP->Control Panel
MySQL->Admin
dort auf Neu und dann eine Datenbank namens "bif1webtechdb_" anlegen. Nun diese auf der linken Seite anklicken und
dann auf importieren und die inkludierte "bif1webtechdb_24.01.14.sql" importieren. 

Diese Datenbank benutzt kein Passwort! Dies sollte auf jeden Fall schnellstmöglich geändert werden. 
Siehe zB. hier:  https://kinsta.com/knowledgebase/xampp-mysql-password/
